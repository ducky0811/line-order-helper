請使用店家自己的官方帳號，通知才會以店家的名義送出。

可在 LINE Official Account Manager 的帳號設定中找到，必須以 @ 開頭。

開啟官方帳號管理
步驟 2

填入兩組連線金鑰

建議使用電腦操作。先按下面的綠色按鈕登入，再照著兩張尋找卡片即可。

前往 LINE Developers 找金鑰 ↗
① 找到 Channel Access Token

選擇店家的 Provider → 點選標示為 Messaging API 的頻道 → 上方選「Messaging API」→ 往下找到 Channel access token → 按 Issue，再複製。

② 找到 Channel Secret

在同一個頻道，上方選「Basic settings」→ 往下找到 Channel secret → 按眼睛或 Show，再複製。

🔒 請貼「Token」與「Secret」，不要填 LINE 登入密碼。本系統會加密保存，設定完成後不會再次顯示。
步驟 3

設定 Webhook

複製網址到 LINE Developers → Messaging API → Webhook URL，開啟 Use webhook 後按 Verify。

步驟 4

綁定接單的店家 LINE
